## Prerrequisitos

Antes de comenzar, asegúrate de tener instalado:

- [Bun](https://bun.sh/) (versión 1.2.4 o superior)
- [Node.js](https://nodejs.org/) (versión LTS recomendada)
- [Git](https://git-scm.com/)

## Pasos de Instalación

1. Clona el repositorio:

```bash
git clone <url-del-repositorio>
cd fullstack-starter
# Recomendado: borrar la carpeta de documentación
rm -rf apps/docs
```

2. Instala las dependencias:

```bash
bun install
```

3. Configura las variables de entorno:

Crea un archivo `.env` en la carpeta `apps/api` con las siguientes variables:

```env
# General Settings
API_PORT=8787
WEB_URL=http://localhost:5173
API_URL=http://localhost:8787

# Authentication Settings
BETTER_AUTH_SECRET=SOME_RANDOM_STRING_MIN_32_CHARACTERS

# Database Settings
DB_FILE_NAME=file:db.sqlite

# Google Settings
GOOGLE_CLIENT_ID=SOME_GOOGLE_CLIENT_ID
GOOGLE_CLIENT_SECRET=SOME_GOOGLE_CLIENT_SECRET

# Encryption Settings
ENCRYPTION_SECRET=SOME_RANDOM_STRING_MIN_32_CHARACTERS
```

## Desarrollo

Para iniciar el entorno de desarrollo:

```bash
bun dev
```

Este comando iniciará todos los servicios usando Turborepo:
- Frontend: [http://localhost:5173](http://localhost:5173)
- API: [http://localhost:8787](http://localhost:8787)
- Documentación: [http://localhost:3000](http://localhost:3000)

## Construcción para Producción

Para construir el proyecto para producción:

```bash
bun build
```
